MotherSoccer
Juego móvil de “futbolín” para dos jugadores, que se van turnando el mismo celular.

http://igdacr.gamejamcostarica.com/mothersoccer/
https://github.com/orlandocr/MotherSoccer

Gameplay
-Player1 ataca (elige una secuencia de pases y remata a marco).
-Player1 le da el celular a Player2.
-Player2 defiende (selecciona rivales para barrérseles y defiende el marco propio).
-Se comparan ambas jugadas para ver si:
  - alguna barrida detiene el avance del balón
  - el portero detiene el remate
  - hay gol
-Ahora Player2 ataca y se repite el ciclo.
